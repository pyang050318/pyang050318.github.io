This project studies intelligent perception and autonomous navigation in an air-ground cooperative scenario.

My work included video data labeling and classification, image super-resolution reconstruction using REAL-ESRGAN, trajectory recognition and tracking using YOLOv8, and preliminary exploration of reinforcement learning for autonomous obstacle avoidance and path planning.

**Keywords:** REAL-ESRGAN, YOLOv8, object tracking, reinforcement learning, autonomous navigation.
