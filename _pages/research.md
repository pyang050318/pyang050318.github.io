# Research Experience / 科研经历

## AI-based Meiotic Spindle Prediction in Human Oocytes  
## 基于人工智能的明场图像人类卵母细胞纺锤体预测

**Role / 角色：** Project Leader / 项目负责人  
**Time / 时间：** Nov. 2025 – Present

This project focuses on AI-based meiotic spindle prediction in human oocytes from bright-field and Hoffman microscopy images. I have been deeply involved in experimental design, model improvement, innovation proposal, video abstract preparation, patent and software copyright writing, and manuscript preparation.

本项目面向明场/霍夫曼显微图像中的人类卵母细胞纺锤体预测问题。我深度参与项目全过程，包括实验设计、模型改进、创新点提出、视频摘要制作、专利与软著撰写以及论文撰写。

**Outcomes / 项目成果：**
- ASRM video abstract accepted, second author  
- Invention patent under application  
- Software copyright under application  
- Manuscript in preparation, co-first author

## Air-ground Cooperative Intelligent Perception Project  
## 陆空协同项目（军工）

**Role / 角色：** Undergraduate Leader / 本科生负责人  
**Time / 时间：** Jun. 2024 – Sep. 2025

In this project, I worked on manually labeling and classifying video data, applying REAL-ESRGAN for image super-resolution reconstruction, improving dataset quality, using YOLOv8 for trajectory recognition and tracking, and exploring reinforcement learning for autonomous obstacle avoidance and path planning.

在该项目中，我负责对网络爬取的视频数据进行手动标注与分类，利用 REAL-ESRGAN 进行图像超分辨率重建以提升数据集质量，使用 YOLOv8 进行轨迹识别与跟踪，并探索基于强化学习的无人车自主避障与路径规划方法。

## RAW-domain Object Detection Based on MindSpore  
## 基于 MindSpore 的 RAW 域目标检测

**Role / 角色：** Second Leader / 第二负责人  
**Time / 时间：** Sep. 2024 – Mar. 2026

This project investigated object detection in RAW-domain images. I was responsible for RAW image conversion and processing, and reproduced YOLOv5-based object detection after converting RAW images into JPG images.

本项目研究 RAW 域图像中的目标检测问题。我主要负责 RAW 域图像的转换与处理，并基于 YOLOv5 复现 RAW 图像转化为 JPG 图像后的识别检测流程。

**Outcome / 项目成果：**
- University-level innovation project accepted

## Micro-project on Object Recognition in Foggy Environments  
## 恶劣环境下目标识别微项目

**Role / 角色：** First Leader / 第一负责人  
**Time / 时间：** Feb. 2024 – Apr. 2024

As the team leader, I led a course research project on object recognition for special vehicles in adverse foggy environments. We investigated and compared different image defogging algorithms and completed research papers on defogging algorithms and object recognition in foggy weather.

作为队长，我带领队员围绕特种车辆在大雾等恶劣环境下的目标识别问题进行调研和实验，完成了《探究不同种去雾算法的优劣》和《特种车辆在大雾天气下的目标识别》等课程论文。

**Outcome / 项目成果：**
- Awarded “Outstanding Micro-project” by the college
